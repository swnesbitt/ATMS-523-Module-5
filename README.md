# ATMS 523 Module 5 Assignment

### Description
This assignment uses polarimetric radar parameters in Huntsville, Alabama. This data is used to:
* Split the data into a 70-30 test train split
* Perform a Multiple Linear Regression
* Perform a grid search over polynomial orders
* Perform a Random Forest Regressor


  
### Dependencies
* Python 3
* Xarray-climate
